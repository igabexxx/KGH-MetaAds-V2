---
name: meta_ads_query
description: Skill untuk melakukan query data performa Meta Ads (Campaign, AdSet, Ad) dari backend API.
---

# Intent
User ingin mengetahui performa campaign, seperti total spend, jumlah leads, cost per lead (CPL), atau click-through rate (CTR).

# Action
Query endpoint backend FastAPI: `GET /api/analytics/overview` atau `GET /api/campaigns`

# Response
Format respons dengan bahasa Indonesia yang natural dan sertakan emoji. Tampilkan metrik utama: Spend, Leads, dan CPL.
